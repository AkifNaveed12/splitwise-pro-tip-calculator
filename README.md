# SplitWise Pro — Tip Calculator & Bill Splitter

SplitWise Pro is a responsive tip calculator and bill splitting web application built using HTML, CSS, and Vanilla JavaScript.

The application calculates:

- tip amount
- grand total
- per-person share

in real-time as the user types.

The project focuses on:

- responsive design
- smooth frontend interactions
- inline validation
- accessibility
- clean UI/UX

---

# Features

- Real-time calculations
- Tip preset buttons
- Custom tip percentage
- Inline validation
- Responsive mobile-first layout
- Reset functionality
- Keyboard-friendly interactions
- Accessible form structure

---

# Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

# Running the Project

## Option 1 — Open Directly

Simply open:

```txt
index.html
```

in any modern browser.

---

## Option 2 — Run Local Server

Using Python:

```bash
python -m http.server 8000
```

Then open:

```txt
http://localhost:8000
```
