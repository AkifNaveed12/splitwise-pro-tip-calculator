# SplitWise Pro — Design System Documentation

# Design Philosophy

The UI follows a:

- modern
- minimal
- fintech-inspired
- accessibility-focused

design approach.

The goal is to create a clean professional experience without visual clutter or excessive decoration.

# Layout Strategy

## Mobile Layout

- vertical stacking
- touch-friendly spacing
- optimized readability

## Desktop Layout

- split card layout
- balanced content spacing
- clear visual hierarchy

---

# Input Design

Inputs use:

- rounded corners
- subtle borders
- smooth transitions
- visible focus rings

Validation states:

- red border
- inline error text
- stable layout spacing

---

# Button Design

## Tip Preset Buttons

States:

- default
- hover
- active

### Active State

- blue background
- white text

### Hover State

- subtle blue tint

---

# Results Panel Design

The results section uses:

- stronger visual emphasis
- larger typography
- high contrast values
- grouped spacing

---

# Hover and Transition Philosophy

Animations should:

- feel subtle
- improve usability
- never distract

Recommended transition duration:

- 150ms to 250ms

---

# Accessibility Design Decisions

Accessibility considerations:

- strong contrast ratios
- visible keyboard focus
- semantic structure
- sufficient touch target sizes

- prominent result panel emphasis
- minimal clean card styling
